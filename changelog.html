v0.1.0 March 2025 Latest
New Features
Video Call AI Avatar — Real-time anime avatar with lip-sync now available in video calls. Supports 40+ languages.
AI Video Generation — Generate up to 30-second videos from text prompts. HD and 4K quality tiers available.
Voice-to-Voice mode — Speak naturally, receive voice responses. Sub-500ms latency with streaming output.
Image Generation v2 — 8 new styles added: Watercolor, Sketch, Cinematic, Fantasy, Architecture, Abstract, 3D Render, Anime.
Admin Panel — Full admin dashboard for managing users, analytics, and system settings.
Public API — REST API for Business plan users. Full docs at /docs.
Improvements
Dashboard redesigned — faster navigation, better conversation sidebar, improved search.
Chat response streaming is now 2x faster on average across all plans.
Language detection now works on first word — no longer needs full sentence to detect language.
OTP verification redesigned with step-by-step visual tracker.
Image generation queue — real-time progress indicator with estimated time.
Bug Fixes
Fixed: Chat history not loading on slow connections.
Fixed: Voice input sometimes misidentifying Urdu as Hindi.
Fixed: Image generation failing silently when prompt contained Arabic characters.
Fixed: Session expiring too early on mobile browsers.
v0.0.9 February 2025
New Features
Image Generation v1 — Text to image with 4 initial styles. Up to 1080p resolution.
Conversation search — Full-text search across all chat history.
Export chat — Download any conversation as PDF or plain text.
Improvements
Signup flow simplified — fewer steps, email OTP now faster.
Mobile web performance improved — first paint time reduced by 40%.
Bug Fixes
Fixed: Long messages being cut off in the sidebar preview.
Fixed: Dark mode flicker on page load.
v0.0.8 January 2025
New Features
Voice input (beta) — First version of voice-to-text input in chat.
Arabic language support — Full right-to-left layout for Arabic conversations.
Keyboard shortcuts — Enter to send, Ctrl+N new chat, Ctrl+/ focus search.
Improvements
Markdown rendering in chat responses — code blocks, bold, lists now display properly.
Response quality significantly improved for Urdu and Hindi prompts.
