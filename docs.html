Quickstart

Get your first response from Nomino in under 2 minutes. You need an API key from your dashboard settings.

Install the SDK

npm install @nomino/sdk

First request

import { Nomino } from '@nomino/sdk'; const client = new Nomino({ apiKey: 'your_api_key' }); const reply = await client.chat.send({ message: 'Explain machine learning simply', language: 'en' // optional — auto-detects }); console.log(reply.content); // AI response text

Authentication

All API requests require a Bearer token in the Authorization header. Get your key from the dashboard under Settings → API Keys.

# All requests use this header Authorization: Bearer nm_live_xxxxxxxxxxxxxxxx

Chat API

POST
/v1/chat/message
Send a message and receive an AI response
GET
/v1/chat/history/{chat_id}
Retrieve full conversation history
POST
/v1/chat/stream
Stream AI response token by token

Request body

{ "message": "Your question here", "chat_id": "optional-existing-chat", "language": "auto", // auto | en | ur | ar | hi ... "max_tokens": 1024 }

Image Generation API

POST
/v1/images/generate
Generate an image from a text prompt
{ "prompt": "A mountain at sunrise, oil painting", "style": "photorealistic", // anime|oil|3d|sketch|cinematic "resolution": "1920x1080", "count": 1 // 1-4 variations }

Error Handling

All errors return a consistent JSON structure with a code and message:

{ "error": { "code": "rate_limit_exceeded", "message": "Too many requests. Retry after 60s.", "status": 429 } }
  • 400 — Bad request, check your parameters
  • 401 — Invalid or missing API key
  • 429 — Rate limit exceeded, back off and retry
  • 500 — Internal error, contact support
